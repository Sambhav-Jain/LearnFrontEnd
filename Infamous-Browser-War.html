Infamous Browser Wars

The web browsers in modern times are supported by almost every device that can connect to the internet. We use web browsers to access information on the World wide web. Mostly, web browsers are used on personal computers, laptops, android and IOS devices, and other operating systems. Today, the browser market share is dominated by Google Chrome web browser. From the simple static HTML browsing to single-page web applications, and, the way web browsers are working on smartphones, the web browsers have come a long way.
The “browser wars”, that started in the early phase of the web for the ultimate dominance in the usage share of browsers. It revolutionised the way the internet works today.

How Web Became What is it Today?

  • Mosaic

    The first web browser, WorldWideWeb was created by Tim Berners-Lee who also invented the World wide web. It was released for the next step in 1991. Further, next year many other browsers for different platforms came out. These browsers were only simple HTML-Viewers software. The web was very different it consisted of very few web pages line after line only text everywhere. They were used by universities to share and read research papers, most used by geeks. This nerd was no other than Illinois computer science student Marc Andreesen.

    He worked at the National Center for Supercomputing Applications (NCSA) at the University of Illinois, where he became familiar with Tim Berners-Lee's open standards for the World Wide Web. Andreessen and full-time salaried co-worker Eric Bina worked on creating a user-friendly browser with integrated graphics that would work on a wide range of computers. The resulting code was the Mosaic Web browser. Having just left Silicon graphics Jim Clark was looking for something new and Mosaic caught his attention. To common people word internet meant nothing but at the same time but he viewed it as a technology for world-dominance. A technology that would revolutionise the way things work in the world, that will bring even opposite poles closer than ever.

  • Netscape

    The popularity of the Mosaic was at the peak, but one of its developers, Marc Andreessen met with Jim Clark of silicon graphics. After he graduated from the University of Illinois, Andreessen and Jim Clark founded Mosaic Communications Corporation in 1994. Because of the conflicts with the University of Illinois, they changed the name to Netscape and thus released the Netscape navigator 0.9, a web browser. The browser was the most advanced available and was an instant success, becoming a market leader while still in beta. Netscape's feature-count and market share continued to grow rapidly after version 1.0 was released. Version 2.0 added a full mail reader called Netscape Mail, thus transforming Netscape from a mere web browser to an Internet suite. During this period, both the browser and the suite were known as Netscape Navigator.

    By the end of 1995, the web boom set by the mosaic was overtaken by the Netscape navigator. The web was receiving a great amount of attention from the public and media, which led Microsoft in the competition. Thus leading to the first browser war. Microsoft vs Netscape

War for dominance of The Web

  • Internet Explorer vs Netscape Navigator

    Having the monopoly at the time, Netscape offered its browser free for non-commercial users but demanded money for a license. The IPO was extremely successful for Netscape and newly added features such as frames, image maps, and javascript iterations were immensely influential. Bill gates realized how the web is going to make a big impact on the future. Microsoft was already dominating the operating system usage share with 90 per cent but Netscape was the one that dominated the web browsers usage share with 80 per cent. That means, if you are buying Microsoft windows, you need to a web browser separately.
    First spark: First, Microsoft sent its engineers to Netscape for discussing the future of Microsoft and the web. Things didn’t go well between Microsoft and Netscape, and after some time, the two companies rarely spoke. Microsoft comes in action as the Netscape’s popularity was increasing and then, just after 15 days of that IPO launch, Microsoft had their browser ready, the Internet Explorer. The Internet Explorer version one was released by Microsoft to challenge the Netscape navigator. Internet Explorer version one was nothing in front of the world-class web browser of that time. Only a little damage was done to the Netscape’s popularity. Because of this, people at Netscape started taking Microsoft’s web plan lightly.

    • Pearl Harbour Attack

      On 7th December 1995, the same day when Japan attacked Pearl Harbor in World War two, Bill gates held a press conference launching Internet Explorer version 2. His intentions were clear, he did not want to enter the market, he wanted to dominate it. "Microsoft Takes War to the Net ", such headlines were coming out. Both of the companies were releasing minor changes every month. The war was going at a rapid pace. One would release a minor change, the second would do the same to give a challenge.

    • 3.0 The Game Changer

      Netscape navigator’s version 3.0 was realized. It included Netscape’s javascript that was greatly influential. It included tables, background-colours, the archive attribute, the applet element, and new HTML tags such as blink and marquee. To counter the new version of Netscape Navigator, Microsoft launched the third version of Internet Explorer. This was the first time in the war when Netscape got serious competition. It included Cascading style sheets(CSS), which later became one of the three core technologies of the web that we still use today. Still, Netscape navigator had ninety per cent usage share of web browsers while Internet Explorer just had a nine per cent of it. The Netscape was still winning the war, but Bill Gates was keen on dominating the web browsing world.

      On 7th October 1997, the tides turned when Microsoft launched the Internet Explorer 4.0. The Internet Explorer 4.0 turned the tide the side of the Microsoft because they integrated the browser with the windows itself.

    • The Last Nail In The Coffin

      The Internet Explorer 4.0 turned the tide in the side of the Microsoft because they integrated the browser with the windows itself. This changed everything. The users didn’t need to get the web browser separately as it came along the Microsoft window. Microsoft already had ninety per cent usage share of the operating system and now the web browser was also coming with it. This greatly reduced installation efforts. The Netscape navigator never recovered and eventually lost the first browser war in 2001 when Internet Explorer 6.0 hit its peak share of ninety-six per cent.

So, Microsoft won this war of web_browser. Though today Google is dominating player in term of web technologies around the world with over 60 per cent share.
